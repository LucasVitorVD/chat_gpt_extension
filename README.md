# Extensão ChatGpt 

## Descrição

Extensão do Chrome utilizando a API da OpenAI para retornar respostas do chat-gpt de forma mais rápida e maleavel. 

## Tecnologias Utilizadas

- React
- TailWind
- Axios
- OpenAI API
